# Travel Maker

여행을 계획하고 커뮤니티를 통해 사람들과 정보를 공유할 수 있는 서비스입니다.

# 실행 방법
1. dump sql 실행

2. springboot project 실행 // 로드 시 필요한 디렉토리 생성

3. vue 프로젝트 내에 .env.local 파일 생성 후 아래 내용 입력
```
VUE_APP_API_BASE_URL="base url"
VUE_APP_KAKAO_MAP_API_KEY="카카오 API KEY"
```
4. vue
- npm install
- npm run serve
